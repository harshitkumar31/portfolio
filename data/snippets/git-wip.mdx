---
title: 'Git WIP'
description: 'A useful alias to check recent branches you have worked on'
logo: 'github.png'
---

You can add the below to `~/.gitconfig` to use this.
```
[alias]
    wip = !git for-each-ref --sort=committerdate refs/heads/ --format='%(HEAD) %(align:35)%(color:yellow)%(refname:short)%(color:reset)%(end) - %(color:red)%(objectname:short)%(color:reset) - %(align:40)%(contents:subject)%(end) - %(authorname) (%(color:green)%(committerdate:relative)%(color:reset))'
```
